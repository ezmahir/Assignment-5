Important FAQ

What is the Document Object Model (DOM)?

A link between JS and HTML. JS considers HTML to be an object of JS and manupulates that way

How do you select an element from the DOM?

We can select any element from the DOM using their tag name, class name or Id.

What is Event delegation in the context of the DOM, and why is it helpful?

Event delegation is a technique in the DOM where a single event listener is attached to a parent element to manage events on its child elements

How do you manupulate an element's attributes and styles using the DOM?

We can manipulate an element's attributes using setAttribute() and styles via the style property in the DOM.
